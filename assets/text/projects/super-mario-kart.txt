<h1>Süper Mario Kart</h1>
<hr>
<p>Süper Mario Kart is an C++ implementation of Super Mario Kart, a kart racing video game developed and published by Nintendo for the Super Nintendo Entertainment System in 1992.<br><br>
Authors: <a href="https://github.com/diegoroyo">Diego Royo</a>, <a href="https://github.com/vmbatlle">Víctor Martínez</a> and <a href="https://github.com/JaviBite">Javier Giménez</a>.<br></p>

<wrap>
<icaption caption="Serious gameplay" image="super-mario-kart/gameplay.gif"></icaption>
<icaption caption="Not-so-serious gameplay" image="super-mario-kart/chaos.gif"></icaption>
</wrap>

<p><br>This game has been built from scratch using the SFML and OpenAL libraries for graphics and audio support. Features a Grand Prix with five tracks (with level editing support) with three difficulties: 50cc, 100cc and 150cc.<br><br>
The purpose of this project is to understand the basic components of a real videogame. The authors do not aim at any commercial profit and release the source code for the general public under GPLv3 license. If anyone from Nintendo reads this, no you didn't.</p>
