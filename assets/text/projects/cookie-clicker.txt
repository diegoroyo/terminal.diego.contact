<p>Cookie Clicker is an idle game about making an absurd amount of cookies. It was created by Julien Thiennot "Orteil" in 2013.<br><br>
I recreated (part of) this game in ROBLOX, adding multiplayer functionality. You could bake cookies with up to 7 friends and compete for the leaderboard. The game is not playable anymore as ROBLOX's API changes have broken it over time.<br></p>
